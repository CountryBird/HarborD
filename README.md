대한민국의 무역 의존도는 2022년 기준 85%에 이르며, 이 중 99.7%가 해상 운송으로 이루어짐.
특히 부산항은 국내 물동량의 29%를 처리하며, 이 과정에서 다량의 오염 물질이 배출됨.

**부산 항만에서 발생하는 대기오염 물질의 배출량과 이의 분포를 정확히 파악하고자 함.**
